Detailed Description

Public API for Retro Rocket video and console output.

Author
Craig Edwards

Provides framebuffer access, console abstraction, ANSI colour control, pixel operations, and Flanterm-backed text rendering. These functions are intended for kernel subsystems and BASIC userland output.
