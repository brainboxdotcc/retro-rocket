Detailed Description

Unified typed expression parser replacing the individual ones for parsing int, double, string.

This file implements a single typed recursive-descent evaluator that understands INT / REAL / STRING and relational + boolean operators.

void up_eval_value ( struct basic_ctx *  ctx,
up_value *  out 
)

Evaluate a single value expression and return its typed result.

Parses one expression using the unified evaluator and writes the result to out without coercion (the kind will be UP_INT, UP_REAL, or UP_STR).

Parameters
ctxBASIC interpreter context; ctx->ptr must point at the start of the expression to evaluate.
[out]outReceives the typed result of the evaluation.
Postcondition
On return, ctx->ptr is positioned at the first token not consumed by the expression. In particular, list separators (','/';'), THEN, NEWLINE, and ENDOFINPUT are left untouched for the caller to handle.
Note
Errors encountered during evaluation are reported via the existing error machinery (e.g. tokenizer_error_print) and reflected in ctx->errored; this function does not throw or longjmp.
