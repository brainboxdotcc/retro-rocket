Compare two memory blocks.

Compares n bytes of the buffers s1 and s2.

Parameters
s1First buffer.
s2Second buffer.
nNumber of bytes to compare.
Returns
An integer less than, equal to, or greater than zero if the first differing byte in s1 is found to be less than, equal to, or greater than the corresponding byte in s2. Returns zero if all compared bytes are equal.
