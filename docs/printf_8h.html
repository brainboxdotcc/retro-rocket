◆ PR_BUFLEN

#define PR_BUFLEN   26

Maximum temporary buffer length for number-to-string conversion.

Largest number handled is 2^64–1. In base 8, this is 21 digits. Add 1 for sign, 1 for NUL terminator, and a few for slop.

◆ fnptr_t

typedef int(* fnptr_t) (unsigned c, void **helper, const void *end)

Output Function Type.

Each printf variant passes characters to a callback function of this type. This decouples formatting from the output device.

Parameters
cCharacter to output
helperOpaque pointer (e.g. buffer write position)
endEnd-of-buffer guard (if bounded)
Returns
0 on success, nonzero if buffer/device is full
