int64_t expr ( struct basic_ctx *  ctx)

Evaluates a full integer expression.

This function parses and evaluates an integer expression from the BASIC program based on the provided context. It supports all standard integer operations.

Parameters
ctxA pointer to the current BASIC program context.
Returns
The result of the evaluated integer expression as a 64-bit signed integer.
