◆ map_keycmp

typedef int(* map_keycmp) (uintptr_t a, uintptr_t b, void *opaque)

Comparator function for keys.

Must return <0 if a < b, 0 if equal, >0 if a > b. The opaque pointer is passed through unchanged to allow custom ordering contexts.
