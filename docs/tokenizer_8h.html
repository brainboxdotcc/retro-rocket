#define GENERATE_ENUM (   NAME,
  ... 
)    NAME,

List of all tokens recognized by the interpreter.

Note that built-in function names are not considered tokens. Instead, they are treated like user-defined functions, but with hardcoded handlers rather than being redirected to user-defined code.

The following #define generates an enum of token types, along with a corresponding array of string representations for each token. These are used in basic.c to perform tokenization during program parsing.
