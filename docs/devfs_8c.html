Register a new text device file in devfs.

Parameters
nameName of the device file (e.g., "debug").
update_cbPeriodic update callback (may be NULL if not needed).
read_cbRead callback (must not be NULL).

The file will appear under /devices/<name>. Names should not contain path separators or exceed the maximum filename length supported by the VFS.
