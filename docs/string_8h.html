◆ gc()

int gc ( basic_ctx *  ctx)

Reset the BASIC temporary string arena.

Reclaims all strings allocated with gc_strdup() by resetting the arena pointer back to its initial position. This does not free memory back to the system; it simply makes the entire arena available for reuse.

Called automatically at the end of each line execution in the BASIC interpreter.

Parameters
ctxBASIC interpreter context containing arena state.
Returns
int Always returns 1.

◆ gc_strdup()

const char* gc_strdup ( basic_ctx *  ctx,
const char *  string 
)

Duplicate a string into the BASIC temporary string arena.

Allocates storage for the given string inside the BASIC interpreter's fixed-size string arena. The returned pointer remains valid until the next call to gc(), which resets the arena.

Unlike kmalloc(), this allocator never frees individual strings; all strings are discarded together when gc() is invoked.

Warning
  • The returned string is immutable. Modifying it (e.g. appending with strlcat or writing past the terminator) will corrupt the string arena and cause undefined behaviour.
  • Empty strings are interned: all calls with "" return the same canonical pointer.
Parameters
ctxBASIC interpreter context containing arena state.
stringNUL-terminated source string to duplicate.
Returns
const char* Pointer to the arena-allocated copy, or NULL on error.

◆ strstr()

const char* strstr ( const char *  haystack,
const char *  needle 
)

Locate a substring in a string.

Scans the string haystack for the first occurrence of the substring needle. The terminating null character is not compared.

If needle is empty, haystack is returned. If needle does not occur in haystack, NULL is returned.

Parameters
haystackNull-terminated string to search.
needleNull-terminated substring to locate.
Returns
Pointer to the beginning of the located substring, or NULL if no match is found.
