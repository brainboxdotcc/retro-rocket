◆ get_weekday_from_date()

int get_weekday_from_date ( datetime_t *  dt)

Calculate the weekday index from a given date.

Uses a modified version of Zeller's Congruence with precomputed tables for month codes and century codes (starting from the 18th century). The function accounts for leap years.

Weekdays are returned as an integer where:

  • 0 = Sunday
  • 1 = Monday
  • 2 = Tuesday
  • 3 = Wednesday
  • 4 = Thursday
  • 5 = Friday
  • 6 = Saturday
Parameters
dtPointer to a datetime_t structure containing the full date
Returns
int Weekday index (0 = Sunday ... 6 = Saturday)
