Detailed Description

Describes an IRQ override from the MADT table.

Supplementary Volume Descriptor (SVD).

Primary Volume Descriptor (PVD).

Directory entry (file or subdirectory).

Compact per-file date entry format (differs from PVD_date).

Date structure as defined in the Primary Volume Descriptor (PVD).

Used to override the default IRQ to GSI mapping, and specify trigger mode and polarity.

Filenames may exceed the fixed filename[12] buffer. Walk the list of entries using length, which accounts for variable-length names.

Defines the metadata for the filesystem. Contains dual-endian fields (LSB/MSB) for cross-platform compatibility.

Placeholder for Joliet or other extensions.
