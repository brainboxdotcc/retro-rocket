Free a previously allocated block.

Parameters
allocAllocator instance.
ptrPointer returned by buddy_malloc.

The block is returned to its owning region’s free list. If its buddy is also free, the two are merged into a larger block. The owning region is identified via a back-pointer stored in the block header.

Initialise a growable buddy allocator.

Parameters
allocAllocator instance to initialise.
min_orderMinimum order (e.g. 6 for 64B blocks).
max_orderMaximum order (e.g. 22 for 4MB region).
grow_orderOrder used when adding new regions (e.g. 22 for 4MB).

The allocator starts with no regions. The first region is allocated automatically on the first call to buddy_malloc, or may be added eagerly by growing manually.

void* buddy_malloc ( buddy_allocator_t *  alloc,
size_t  size

Allocate a block of memory from the buddy allocator.

Parameters
allocAllocator instance to allocate from.
sizeRequested size in bytes.
Returns
Pointer to allocated memory, or NULL if out of memory.

If the active region cannot satisfy the request, a new region is automatically allocated of size (1 << grow_order). Returned blocks are aligned to their block size (power of two).

Reallocate a block within a buddy allocator.

Works like krealloc(): if ptr is NULL, behaves like buddy_malloc(). If size == 0, frees the block and returns NULL.

Parameters

Returns
Pointer to resized block, or NULL on OOM.
