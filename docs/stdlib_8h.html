Detailed Description

Minimal standard library stubs for Retro Rocket.

Author
Craig Edwards
Date
2023-05-27
Copyright
Copyright (c) 2023

Provides a reduced subset of C standard library functions. Many functions are provided only as stubs for compatibility. Multibyte and wide-character conversions are not supported.
