Detailed Description

Represents a single handler in a chain of handlers attached to a shared ISR.

Some IRQ lines may be shared between multiple devices (especially in legacy or MSI-less systems). This structure forms a linked list of handlers associated with a given interrupt vector. Each node contains the handler function, the originating PCI device (used to support IRQ sharing), and an opaque pointer passed to the handler at runtime.
