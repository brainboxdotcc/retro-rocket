void sockread_statement ( struct basic_ctx *  ctx)

Process SOCKREAD statement.

The SOCKREAD statement will yield while waiting for data, essentially if there is no complete line in the data buffer yet, it will yield back to the OS task loop for other processes to get a turn. Each time the process is entered while waiting for the data to be completed, it will just loop back against the same SOCKREAD statement again until completed.
