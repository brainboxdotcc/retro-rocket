◆ kfree_null

#define kfree_null (   p)    do { kfree(*(p)); *(p) = NULL; } while (0)

Free memory and set the pointer to NULL.

This macro safely frees a dynamically allocated pointer and sets it to NULL to avoid dangling references. It must be passed the address of the pointer (i.e., a pointer to the pointer).

Example usage:

void* ptr = kmalloc(128);
kfree_null(&ptr); // ptr is now NULL
