Detailed Description

Per-device block cache.

A cache instance attached to a storage device. Holds a bounded number of sector entries in a hashmap for O(1) lookup and in an LRU queue for O(1) eviction. Tracks recency using a monotonically increasing tick counter. All access is synchronised by a spinlock.
