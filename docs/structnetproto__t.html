Detailed Description

Represents a protocol, or group of related protocols, that can be attached to an interface and have addresses.

Currently only Ethernet devices are supported, covering the majority of consumer hardware. Protocols may have dependencies (e.g. ARP required by IPv4), which are added automatically when needed.
