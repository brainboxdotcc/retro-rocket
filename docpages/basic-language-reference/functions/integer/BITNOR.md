\page BITNOR BITNOR Function
```basic
X = BITNOR(a, b)
```

Computes the **bitwise NOR** of two integers.

---

### How to read it
- Equivalent to `NOT (a OR b)`.

---

### Examples
```basic
PRINT BITNOR(&6, &3)
```

---

### Notes
- Operates on 64-bit integers.
