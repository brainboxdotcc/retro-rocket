\page BITNAND BITNAND Function
```basic
X = BITNAND(a, b)
```

Computes the **bitwise NAND** of two integers.

---

### How to read it
- Equivalent to `NOT (a AND b)`.

---

### Examples
```basic
PRINT BITNAND(&6, &3)
```

---

### Notes
- Operates on 64-bit integers.
