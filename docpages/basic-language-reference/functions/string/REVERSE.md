\page REVERSE REVERSE$ Function
```basic
REVERSE$(string-expression)
```
Reverses string and returns the result
## Examples
| Input            | Output           |
|------------------|------------------|
| ``abcdef``       | ``fedcba``       |
| ``0123456789``   | ``9876543210``   |
| ``hello world!`` | ``!dlrow olleh`` |
| ``Retro Rocket`` | ``tekcoR orteR`` |
| ``BASIC``        | ``CISAB``        |
