\page BITOR BITOR Keyword
```basic
X = BITOR(a, b)
```

Computes the **bitwise OR** of two integers.

---

### How to read it
- Returns all bits set that are set in `a` or `b`.

---

### Examples
```basic
PRINT BITOR(&5, &3)
```

This example produces `7`.

---

### Notes
- Operates on 64-bit integers.
