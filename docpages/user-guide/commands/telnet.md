# Telnet Client

```
telnet <hostname> [port]
```

The telnet client allows you to connect to telnet servers. If you do not specify a port number on the command line, the default port number for telnet (23) is used.

\image html telnet.png

## Exiting telnet

To exit gracefully, press `ESC`. To force the program to exit, you may press `CTRL+ESC` as with all BASIC programs.
